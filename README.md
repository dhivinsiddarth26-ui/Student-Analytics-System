# Student Analytics System

The Student Analytics System is a Python-based interactive dashboard developed using Streamlit to analyze and visualize student data efficiently. It connects to a database using SQL to retrieve structured information such as student ID, name, age, city, and course details, and uses Pandas for data processing and analysis. The application presents key insights including average student age and city-wise distribution through tables and visualizations, providing a clear and user-friendly interface for exploring the data. This project demonstrates practical skills in data analysis, database integration, and dashboard development, while also following modular coding practices and version control using Git, making it suitable for inclusion in a professional portfolio.

## Features
- View student dataset
- Calculate average age
- Visualize students per city

## Run the Project
pip install -r requirements.txt  
streamlit run app.py
